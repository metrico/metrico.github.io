## Installation Script
### DEB System
```bash
wget -q -O - https://github.com/metrico/metrico.github.io/raw/main/libchdb_installer.sh | sudo bash
sudo apt install -y libchdb
```
### RPM System
```bash
wget -q -O - https://github.com/metrico/metrico.github.io/raw/main/libchdb_installer.sh | sudo bash
sudo yum install libchdb
```
